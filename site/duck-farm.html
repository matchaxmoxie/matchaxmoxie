因 why

restaurant-kid wiring

restaurant uptime taught the same lesson as debugging: ship what works, not what sounds good in a pitch. the virtual duck farm is a small, honest react sandbox for that instinct. click the grass. place ducks. tap one and it jumps. then run the konami stress-test (↑↑↓↓←→←→BA) and watch fifty ducks stampede without melting the main thread.

the duck never got a seat in the zodiac. we gave it one anyway. ducks are loud, loyal, and you can't catch one unless you know the pond. the pond lives on the briefcase.

栈 stack

honest stack

no d3. no tailwind. the farm is react + typescript + vite, deployed to github pages as a portfolio feature page with an architecture narrative under the grass.
