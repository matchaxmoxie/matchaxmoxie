AI Transparency

Cursor, Agentic, Copilot

Golden Dog Theory: explicit process, formal verification, and human accountability.

Instrumenta (Tooling Scope)

I use Cursor, agentic assistants, and GitHub Copilot for drafting, refactor support, ideation, and iteration velocity.

These tools increase iteration speed; they do not replace ownership, architectural judgment, or release authority.

Golden Dog Logic Model

  1. Input Control: prompts include constraints, context, and expected outputs.
  2. Draft Generation: AI proposes code and text alternatives for comparative evaluation.
  3. Verification Gate: outputs are evaluated for logic, policy fit, and accessibility conformance.
  4. Traceability: major decisions are documented so rationale remains inspectable.
  5. Human Sign-off: final publication requires explicit human authorization.

Theoria (Execution Discipline)

PB sandwich CS theory is the operational rule: if instructions are vague, outputs become unstable.

Prompting and review follow algorithm design: define inputs, state assumptions, order steps, test edge conditions, then iterate.

Transparency is part of quality assurance. Tool usage is disclosed so the workflow is auditable.
