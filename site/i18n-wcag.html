language switching is not decoration. restaurant floors and immigrant households already change mid-sentence. if the DOM pretends everything is one language, screen readers and layouts both fail.

what to remember

  • mixed-language DOM: lang on root and on inlined 中文 / other scripts so assistive tech enunciates correctly.
  • character density: english labels are not zh-width. containers wrap; truncation is an a11y decision.
  • roots without performative gloss: 福州话 / mandarin / english at home → software that does not assume monolingual default.
