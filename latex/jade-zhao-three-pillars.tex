\section*{Three gentle anchors}

\begin{enumerate}
    \item \textbf{People \& warmth}

    Big schools can feel lonely. I like work that makes the path kinder: clearer resources for first-gen and low-income friends, ServeIT sprints where the client feels heard, study-abroad forms I have cried over too.

    \item \textbf{Building \& fixing}

    Quiet pride when something works outside a screenshot: HIPAA-ish data chores, monitoring that helps night shifts, Phenology plots that tell the truth, civic sites that load for real humans.

    \item \textbf{Access \& care}

    WCAG as a habit, not a panic. Educators still inside AI loops. If only some people can finish the task, we are not done yet.
\end{enumerate}
